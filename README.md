# Greenwood International Academy — School Website

A complete, modern, production-style school website built with **Python**, **Django**, **HTML**, **CSS**, **Bootstrap 5**, **JavaScript**, and **Chart.js**.

This project is designed for **Greenwood International Academy** and includes both a public-facing school website and an enhanced Django admin panel for managing data.

---

## Live Project Overview

This website includes:

- Responsive multi-page school website
- Enhanced Django admin panel
- Admission enquiry system
- Fee payment form with receipt preview
- Faculty showcase
- Notifications module
- Contact form
- Gallery page
- Dashboard / statistics page with charts
- Realistic dummy data
- Mobile, tablet, laptop, and large-screen responsive layout
